# Portfolio Project

This is a Turborepo monorepo containing a Next.js 15 portfolio website with an integrated blog system.

## Architecture

- **Framework**: Next.js 15 with App Router and React 19
- **Content**: Database-backed MDX with next-mdx-remote for compilation
- **Database**: Neon PostgreSQL with Drizzle ORM
- **Cache**: Upstash Redis for analytics and caching
- **Styling**: Tailwind CSS v4
- **Testing**: Vitest with React Testing Library
- **Code Quality**: Biome for linting/formatting

## Structure

- `apps/blog/`: Main Next.js application
- `packages/`: Shared packages (currently empty)
- Root contains Turborepo configuration and shared tooling

## Key Features

- Custom privacy-focused analytics with IP hashing
- MDX-powered blog with reading time calculation
- Real-time visitor statistics dashboard
- Performance optimized with caching and image optimization
- SEO with structured data and OpenGraph image generation

## Development

- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm test` - Run tests
- `pnpm lint` - Run linting

## Database

Schema in `apps/blog/db/schema.ts` using Drizzle ORM with session tracking for analytics.
