# Blog Application

This is the main Next.js application for the portfolio website.

For complete documentation, see the root-level [README.md](../../README.md) and [CLAUDE.md](../../CLAUDE.md).

## Quick Start (from this directory)

```bash
bun install          # Install dependencies (run from root)
bun run dev          # Start dev server
bun run test         # Run tests
bun run build        # Build for production
```

For all available commands and detailed documentation, see [../../CLAUDE.md](../../CLAUDE.md).
